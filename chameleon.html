    Engagement Platform

    Chameleon is a cloud-based platform for internal communications and training.

    Chameleon clients use it in a SaaS (Software as a Service) model, as well as directly utilizing a Chameleon team of designers and support.

    Chameleon uses a responsive (desktop/tablet) and adaptive (mobile) framework with customizable pages and interactions to engage and assess the employee's knowledge. Clients can update text, media, pages with a cloud-based course editor. It's hosted on Amazon's S3 server, so global access is seamless and can reach anywhere the internet can go.

    Art Direction + UX Design + Product Design

    Platform Overview

    Courseware
    A module or course that the end user completes. It has unique page interactions and media objects keep things interesting. Pages and media objects in the course are responsive on desktop and tablet sizes.
    Editor
    Where courses are created. Roles can be assigned to different users, so different parts of courseware can be seen by certain groups. Themes can be created once to apply styles and assets to multiple courses.
    Adaptive Courseware for Mobile
    Slimmed down interactions for mobile to simplify the experience, as well as image handling tools so users can inspect large images inline in the courseware.
